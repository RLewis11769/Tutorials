# Components

## Description

Practice/guide for how to create site components using jQuery

## Navigation

### 1. slider

#### Description

- Essentially what's going on behind the scenes in slideshows
- Keeps it simple by setting the height/width of each pic and hardcoding the width in the JS
- Uses images/ directory to store pics

#### Features

- Cycles through 4 pics
- Caches DOM to enhance performance
- Pauses slideshow on hover

### 2. panel

#### Description

- Essentially what's going on behind the scenes in two-level tabs
- Hides and shows panel content on click

#### Features

- Uses find for easy way to locate specific element
- Caches DOM to enhance performance
